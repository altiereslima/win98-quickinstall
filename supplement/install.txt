Your friendly Linux-based Windows 98 / ME Installer :)

Features:
- This will install Windows 98 / ME in RECORD time.
- Can include many patches and slipstreamed drivers
- Can boot from floppy, CD-ROM, USB, hard disks, ...
- Writes MBR and boot sectors for FAT32/FAT16
- Can partition your hard drive(s)

Bugs:
- Error handling is basically nonexistant. Sorry!
